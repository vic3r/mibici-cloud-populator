# Mibici cloud populator


### Prerequisites
```
Node => 8.11
```
Node can be installed using NVM running 
`nvm install 8.11`

```
yarn
```

yarn can be installed using brew
`brew install yarn`

### Run
Install dependencies
```
yarn install
```

Overwrite `.env` file with your own environment variables

In dev ennvironment

```
yarn start-dev
```

In prod environment
```
yarn start
```
